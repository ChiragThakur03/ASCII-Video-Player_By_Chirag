# ASCII Video Player 🎬

A powerful CLI tool to transform any video into vibrant, colored ASCII art animations. Experience 24-bit True Color playback directly in your terminal or export your creations as stylized MP4 videos.

---

## 🌟 Key Highlights

- **Terminal Cinema:** Watch videos converted to ASCII in real-time with full color support.
- **MP4 Export:** Save your ASCII animations as permanent video files with customizable backgrounds.
- **Engineered for Performance:** Optimized with multithreading and NumPy for smooth frame rendering.
- **User-Centric Design:** Guided CLI flow with persistent branding and intuitive configuration.
- **Customizable:** Control output width, frame skipping, and aspect ratio correction.

## 🚀 Quick Run
```bash
python ASCII_V1.py your_video.mp4
```

## 🛠️ Tech Stack
- **Python 3.7+**
- **OpenCV:** Video processing and codec management.
- **NumPy:** Vectorized color and character mapping.
- **Pillow:** Image generation for high-resolution exports.
